# Helyn Logger for C# - WIP

A custom logger for Unity.

## Status
WIP

## Main goal
The main features I want for this logger are:
- filter logs using namespace, class name, and log level
- have a custom log file
- have custom log formats for console and log file
- have colors for log levels in Unity console
- use default Unity logging classes
- catch and filter all logs, including Unity native and third party
