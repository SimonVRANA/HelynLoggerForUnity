## Install
To install this package in Unity, you have two options:
- Add `"com.helyn.unity.helynlogger": "https://github.com/SimonVRANA/HelynLoggerForUnity.git?path=com.helyn.unity.helynlogger"` to your `Packages/manifest.json` file.
- Go to Window > Package Manager > v + (top-left) > **Add package from git URL...** and paste: `https://github.com/SimonVRANA/HelynLoggerForUnity.git?path=com.helyn.unity.helynlogger`.
